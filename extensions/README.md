## 🔵 ES7 React/Redux...
Ezt nagyon ajánlom! Órákon is használtuk Reacthoz! A képernyőképemen láthatod hogyan kell telepíteni.

![Es7 Extension](https://i.ibb.co/jPG5R2Qg/es7.gif)

Rengeteg előre beépített rövidítés van benne. Itt van néhány példa:

`rafce` ---> React Arrow Function Component with Export-ot ad vissza.

```
import React from 'react'

const App = () => {
  return (
    <>
    </>
  )
}

export default App
```

`clg` ---> Console Log-ot kapunk.
```
console.log(*ide mehet amit logolni szeretnél*)
```
